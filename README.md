# GraniteDB

**GraniteDB** is a blazing-fast, document-oriented NoSQL database designed for modern developers.   It combines MongoDB-like flexibility with next-gen performance, built from scratch in Rust.

---

### 🚀 Features (WIP)

- Document-based data model (JSON-like)
- Smart querying engine
- Aggregation pipelines
- Write-ahead logging (WAL)
- Native encryption at rest
- Role-based access control
- Horizontal scaling (sharding-ready)
- Developer-first CLI & dashboard
